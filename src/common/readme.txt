Папка common містить файли спільні для обох редакцій PowerCab: для FX і GD. Файли зберігаються в одному екземалярів в репозиторії, щоб уникнути дублювання коду. У розробників чи в деплоєній версії такі файл дублюються. Синхронність файлів підтримується утилітою sync.js (див. папку tools).

Файл Info.xml містить певні поля, які описують проект, але загальні для обох редакцій, вони також переписуються у відповідні сирці відповідних редакцій за допомогою утиліти sync.js.
